TGCC
====

Dies ist ein Projekt im Rahmen des Turtle Graphics Creativity Contests im Rahmen der Veranstaltung SE-1.

Zielsetzung
====

Mithilfe der Möglichkeiten aus Level 1 und 2 soll eine möglichst kreative Lösung entwickelt werden.

"Höhere" Java-Funktionen wie Arrays, HashMaps und dergleichen dürfen noch nicht benutzt werden. Auch darf keine Vererbung benutzt werden.

Abgabekriterien
====

Die Abgabe erfolgt in einem gezippten BlueJ-Paket. Zudem muss mindestens eine Schleife programmiert sein und Lösungen mit Rekursion werden bevorzugt.

Start
====

Das Projekt wird mit der Methode start() in der Klasse Main gestartet.

Umsetzung
====

Dieses Projekt stellt ein Dame-Spielfeld dar mit ein paar Zügen.
